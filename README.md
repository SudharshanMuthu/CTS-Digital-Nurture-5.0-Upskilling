## About the Repository

This repository documents my learning journey and hands-on implementations completed as part of the **Cognizant Digital Nurture 5.0 Upskilling**.

The repository contains module-wise exercises, practice programs, mini-projects, and learning artifacts developed while exploring modern web technologies, database systems, and Java programming concepts.

The primary objective of this repository is to demonstrate continuous learning, practical implementation, and structured progress tracking throughout the upskilling journey.

---

## Learning Methodology

Throughout the program, I adopted a structured and practice-oriented learning approach:

* Learned the core concepts before implementation.
* Practiced each topic through guided exercises.
* Solved hands-on problems independently.
* Built mini-projects to apply concepts in real-world scenarios.
* Maintained clean and organized source code.
* Used Git and GitHub for version control and progress tracking.
* Focused on writing readable, maintainable, and structured code.

This approach helped strengthen both conceptual understanding and practical software development skills.
