Bootstrap Structure and Files

CSS Folder

bootstrap.min.css contains all Bootstrap styling for buttons, forms, tables, grid layouts and responsive design.

JS Folder

bootstrap.bundle.min.js contains Bootstrap JavaScript plugins like Modal, Accordion, Dropdown and Collapse.

Icons Folder

Bootstrap Icons are used to display ready-made icons such as search, menu and social media icons.
